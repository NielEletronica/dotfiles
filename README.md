# i3 Arch Dotfiles

## Install

git clone https://github.com/NielEletronica/dotfiles
cd dotfiles
chmod +x install.sh
./install.sh

## Includes

- i3wm
- alacritty
- dunst
- rofi
- pywal theme
- archlinux-wallpaper integration

## Theme

- Colors: gritty.png (pywal)
- Wallpaper: svalbard.jpg

## Backup

./backup.sh
